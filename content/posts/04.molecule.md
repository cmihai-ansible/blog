## Test using the default engine (docker)

```bash
molecule create
molecule converge
molecule verify
molecule login
molecule destroy

# or:
molecule test
```

## Test on KVM:

```bash
molecule test -s kvm
```

## Test on podman:

```bash
molecule test -s podman
```

## Change the OS:

```bash
export CONTAINER_COMMAND="/usr/sbin/init"
export CONTAINER_DISTRO=centos
export CONTAINER_VERSION=7
molecule --debug test
```
